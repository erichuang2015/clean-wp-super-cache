This plugin can be used along with WP Super Cache, this plugin allows you to clear   Cache from any page with single click without leaving that page

== Description ==
This plugin adds one feature to WP Super Cache.

If you have used WP Super Cache you must be knowing how difficult it is to clear cache in it, you are working on some article and then you want to test it on the front end you have to open new tab clear cache and then you can see.

But this plugin adds a button on top of every page of Admin bar from there you can clear out complete cache without leaving the page, It deletes the Cache using Ajax

Here is a Video to help you out in Installation and usage of the plugin

[youtube https://www.youtube.com/watch?v=evXYw-yjiBY]

== Installation ==
Step 1: Download the latest copy of the Plugin or Search and install directly from WordPress repository

Step 2: Install the plugin like regular plugin

Step 3: That is it, now you will add a button on top of the Admin bar as \"Clear all Cache\" 

Step 4: Click that button and it will clear all the Cache created by WP super cache

== Frequently Asked Questions ==
I installed the plugin but I don\'t see the Button on top of my dashboard
Ans:  Make sure WP Super Cache is installed and Enabled on your site, And this button will only appear on the backend
